# CellSentry v2 Python dependencies
#
# v2.0 ships with OCR DISABLED by default. Modern Chinese hospital records are
# digital PDFs and DOCX exports — pdf-parse + mammoth.js handle those without
# Python. For the rare image-input case, users can extract text via system OCR
# (macOS Live Text / Windows OCR API) and paste, which is on par with our
# OCR engine for standard printed Chinese (per D31 + 2026-04-28 audit).
#
# These deps only matter if the user opts into image OCR via:
#   CELLSENTRY_OCR_TIER={bf16|8bit|4bit|ds-ocr-2} npm run dev
#
# Install when you actually need image OCR:
#   pip install -r requirements.txt
#
# Then download via app first-launch flow.
